### Clone this repo
```bash
git clone https://github.com/abroudoux/hostels-management-app
```

### Install dependencies
```bash
composer install
```

```bash
npm run build
```

### Configure your environment variables in `.env`

```bash
cp env.example .env
```
Don't forget to setup your db then connect to your project with:

```bash
php artisan db
```
### Run the project
```bash
php artisan serve
```
